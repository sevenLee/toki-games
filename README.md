# Purpose
For TOKI GAMES Test

### Install Yarn at first time
```bash
$ curl -o- -L https://yarnpkg.com/install.sh | bash
```

### Start
```bash
$ yarn install
$ yarn start
``` 
You can now view the project on `http://localhost:3000`
